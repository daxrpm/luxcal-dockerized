Help for Upcoming view

In this view you will see a list with upcoming events. At the top of the page the time span can be selected. Initially the start date will be the date of today and the end date depends on the time to look ahead specified by the administrator.

On this page you will find the following hyper-links:
  • Event title will open the Event Report with all event details, or, if you have edit rights, it will open the Event Edit window.
  • Two download buttons, one for a text file and another for a CSV file, to download a file with all upcoming events shown on this page. For the CSV file the separator will be the vertical bar ( | ) character.

For an explanation of the Navigation bar buttons, select the General Help button in the right upper corner.
