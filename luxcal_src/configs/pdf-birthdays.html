1. Introduction

In the LuxCal calendar the PDF Birthday Print feature can be enabled on the administrator's Settings page in the section Navigation Bar. When enabled, an option "PDF File - Birthdays" will appear in the calendar's side-menu.

When the user selects the PDF File - Birthdays option in the side menu, a PDF dialog is opened where the user can specify the PDF print format (Portrait or Landscape). Active selections in the calendar (e.g. via the Options Panel or restrictions via the user group) are also applied by the PDF print feature.

Details on how to populate the birthday calendar can be found in the admin_guide.html file.

2. Tailoring the PDF's configuration settings

The PDF Birthday Calendar creation scripts have a section with default configuration settings, defining the graphical user interface style.

You can change the default PDF configuration settings by copying the file pdfbc.cnf in the "samples" folder to the 'configs' folder and and by tailoring the settings in this file.

3. Explanation of the PDF configuration settings

  • $calID = "";
    Calendar ID. If there are multiple calendars in use, here you can specify which calendar to use. E.g. "mycal". When this setting is left blank (""), the default calendar will be taken.
  • $pageFtr = "#c>>#d";
    Page footer template (#c: calendar title, #d: date printed, >: tab left > center > right)
